# Booking System

## Description
This project is a full-stack booking system using Node.js, Express, and MongoDB. Users can register, log in, view services, and create bookings.

## Technologies
- Node.js
- Express
- MongoDB
- JWT
- HTML, CSS, JavaScript

## Database Collections
- users
- services
- bookings

## API Endpoints
Authentication:
- POST /auth/register
- POST /auth/login

Services:
- GET /services
- POST /services

Bookings:
- GET /bookings
- POST /bookings

Aggregation:
- /bookings/stats/by-service
- /bookings/stats/revenue

## How to Run
1. npm install
2. node server.js
3. Open frontend/index.html
